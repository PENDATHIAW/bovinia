# Logo BOVINIA

Placez ici le fichier logo officiel :

```
bovinia-logo.png
```

**Format recommandé :** PNG fond ivoire transparent ou ivoire mat, haute résolution (min. 800 px de large).

Le logo officiel comprend :
- Monogramme **B** (tête de bœuf + feuille)
- Nom **BOVINIA** (le **I** en forme d'os)
- Tagline **POWERED BY BONE BROTH**
- Séparateur avec icône bol

**Variantes optionnelles** (pour évolutions futures) :
- `bovinia-logo-icon.png` — monogramme B seul (favicon mobile)
- `bovinia-logo-white.png` — version pour fonds sombres

Le site référence : `/assets/logo/bovinia-logo.png`
